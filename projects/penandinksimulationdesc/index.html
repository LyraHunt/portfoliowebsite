Pen and Ink Simulation

(2024)

A coding project that simulates the style of pen and ink on paper.

Made with Javascript and HTML Canvas.

Inspired by this reddit post, but this is my own implementation.

This is a coding project that I made to study the look of pen and ink on paper. The main part of the program is a function that takes two positions and then draws a line between them. It traces between the two points and places small circles of varying sizes along the line. The program involves a lot of random functions, because randomness introduces imperfections which make it feel more natural or even 'human'.
